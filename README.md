->Spotify Music Analysis

A beginner-friendly data analysis project exploring **Spotify music trends** — artist popularity, track duration, and correlations between features.

->Project Overview
This project uses Python (Pandas, Matplotlib, Seaborn) to analyze a dataset of Spotify tracks.  
It covers:
- Cleaning and exploring the dataset  
- Finding the most popular artists  
- Analyzing correlations between musical features  
- Visualizing insights using heatmaps and scatter plots  


->Dataset
Dataset: [Spotify Tracks Dataset on Kaggle](https://www.kaggle.com/datasets/yamaerenay/spotify-dataset-1921-2020-160k-tracks)


-> Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook / Google Colab  

->Sample Output
Includes:
- Correlation heatmap  
- Scatter plot showing popularity trends
